steps to create a GitHub account:

1. Creating a GitHub account
2. Go to the GitHub website (https://github.com/) and click on the "Sign up" button in the top right corner of the page.
3. Enter your email address, username, and password in the sign-up form.
4. Click on the "Sign up" button to create your account.
5. You will receive a confirmation email. Click on the link in the email to verify your account.
6. Once you have verified your account, you can log in to GitHub using your email address and password.


create a new repository in GitHub:

1. Go to GitHub and sign in to your account.
2. Click on the '+' icon in the upper right corner and select 'New repository' from the dropdown menu.
3. In the 'Repository name' field, type a name for your new repository which is 2202030400151.
4. You can choose to make the repository either public or private. Public repositories are visible to anyone, while private repositories are only visible to you and any collaborators you invite.
5. If you want to add a README file, description, or .gitignore file, you can do so in the 'Add files' section.
6. Once you've filled out the necessary information, click the 'Create repository' button.
